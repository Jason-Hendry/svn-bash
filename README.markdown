Functions Provide

#### Add, Update and Create module

**modadd** <mod> <version>
**modup** <version>
**modnew** <mod>

#### Switch between Read-Write and Read-Only versions

**modrw** <version>
**modro** <version>

#### Get information about module version

**modlatest**
**modversion**
